# Motion Magnification

C++ implementations of the 
[Eulerian Motion Magnification](http://people.csail.mit.edu/mrub/vidmag/)
algorithm.

### Dependencies 

- OpenCV
- OpenGL for visualization
- OS X / Cocoa for UI

### Example
Plot pulse rate from image intensity.
